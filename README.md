# Wiam's Oil Store

نسخة متجر مرتبطة بـ Supabase وفيها:

- صفحة متجر
- سلة مشتريات
- Checkout
- حفظ الطلبات في Supabase
- صفحة `/admin.html`
- تسجيل دخول للأدمن
- إدارة منتجات
- إدارة طلبات
- تعديل إعدادات المتجر
- رفع صور إلى Supabase Storage

## 1) متغيرات البيئة في Vercel

بدل المتغيرات القديمة، أضف هذه القيم:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_KEY`

> هذه النسخة مبنية بـ Vite، لذلك لازم تكون الأسماء `VITE_...`

## 2) لازم تعمل مستخدم أدمن

من Supabase:
- Authentication
- Users
- Add user

## 3) شغّل ملف SQL

افتح `supabase-setup.sql` في SQL Editor داخل Supabase وشغّله بعد ما تغيّر:
- `your-email@example.com`

إلى إيميل الأدمن الحقيقي

## 4) استبدال ملفات GitHub

احذف الملفات القديمة من الريبو وارفع هذه الملفات كلها بدلها.
ثم اعمل Redeploy من Vercel.

## ملاحظة مهمة
إذا ما شغّلت ملف SQL، رح يشتغل جزء من المتجر لكن:
- حفظ إعدادات المتجر لن يعمل
- الحماية الحقيقية لن تكتمل
- رفع الصور قد لا يعمل
